[팀장(깃 관리자)]
1. github에 저장소 만들기
2. 작업을 위한 프로젝트 폴더를 이클립스에 만들기
3. 로컬 저장소를 만들기
4. 스테이지 영역에 github 저장소에 올릴 파일들을 추가하기
5. 작업 디렉토리에 있는 파일을 로컬 저장소에 저장하기(commit)
6. remote 연결하기
	-git 원격 저장소에 연결하기 위해서 토큰을 발급(한 번 발급하면 다시 github에서 확인 불가)
7. remote로 push
	-pull request
	-confirm
8. main 브랜치로 동기화
	- 이클립스에서 main브랜치 생성
	
	
	
	
[개발을 위해서 팀원이 작업해야 할 일]
1. step1
	1) 팀장징의 계정을 검색하고 작업할 프로젝트를 fork
	2) fork한 프로젝트를 이클립스에서 프로젝트로 등록
	미션
	first 프로젝트를 팀원계정에서 포크하고 이클립스에 연결
	팀원 깃허브 저장소 목록 캡처
	깃워크2 이클립스 뷰 캡처

2. step2
	1) 작업브랜치 추가하기
	2) 나의 작업 추가하기
	3) 스테이징 영역에 추가(add)
	4) 로컬저장소에 추가(commit)
